AnimeVerse 🎌

AnimeVerse is a responsive anime-themed website built using HTML, CSS, and JavaScript. 
The project demonstrates modern front-end developmentconcepts, including responsive
layouts, image sliders, navigation, and form validation.

📂 Project Structure
Anime/
├── home.html
├── newnav.html
├── popular.html
├── Signin.html
├── login.js
├── script.js
└── style.css

Features
	Home Page
o	Responsive navigation bar
o	Hero section
o	Anime-themed design
o	Image slider/carousel

	Popular Page
o	Displays popular anime content
o	Responsive layout

	New Released Page
o	Dedicated page for new anime
o	Clean and organized UI

	Sign-in Page
o	Email validation
o	Password validation
o	Error messages for invalid input
o	Success message after valid login
o	Success message automatically disappears after 4 seconds
o	Form automatically resets after successful login
o	Google and Facebook login button placeholders

	Technologies Used
o	HTML5
o	CSS3

	JavaScript Concepts Used
o	DOM Manipulation
o	Event Listeners
o	Form Validation
o	Regular Expressions (Regex)
o	Functions
o	Conditional Statements
o	setTimeout()
o	classList.add() and classList.remove()

	How to Run
o	Clone this repository.
o	Open the project folder.
o	Open Anime/home.html in your browser.
